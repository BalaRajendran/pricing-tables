Requirements

  • A working web server (Recommended Apache)
  • PHP 5.4 Upwards (Recommended 5.6+)
  • A SCSS watcher if you want to edit the SCSS files

Setup

  1. Place the pricingtables folder onto your web server
  2. View the project by loading index.php.

Developer Notes

  1. Loading specific tables - Each Pricing table is dynamically loaded. For each pricing table, all you need to load is the relevant php page e.g. pt-card.php in the base directory and then the matching CSS file e.g. assets/css/pt-card.scss along with the base code inside assets/css/style.scss.
  2. SCSS for styling - This project uses SCSS to power the CSS file, it's basically intelligent CSS. For more information on how to use it, click here.
